# ⏱️ ストップウォッチアプリ

シンプルで使いやすいWebベースのストップウォッチアプリケーションです。

## 🌟 特徴

- ⏱️ **高精度計測**: 100分の1秒まで正確に計測
- 📊 **ラップタイム機能**: 複数のラップタイムを記録可能
- 🎨 **モダンなUI**: 見やすく美しいグラデーションデザイン
- 📱 **レスポンシブ対応**: PC・タブレット・スマートフォンで利用可能
- 🚀 **軽量**: 外部ライブラリ不要、1ファイルで完結
- 🌐 **オフライン動作**: インターネット接続不要

## 🚀 使い方

### ローカルで実行

1. リポジトリをクローン
```bash
git clone https://github.com/ponpopko/TEST2.git
cd TEST2
```

2. `index.html` をブラウザで開く
   - ダブルクリックで開く
   - または右クリック → 「プログラムから開く」→ ブラウザを選択

### GitHub Pagesで公開

1. リポジトリの「Settings」→「Pages」にアクセス
2. Sourceで「main」ブランチを選択
3. 保存後、数分で `https://ponpopko.github.io/TEST2/` でアクセス可能

## 🎮 操作方法

### ボタン説明

| ボタン | 機能 |
|--------|------|
| **開始** | ストップウォッチを開始します |
| **停止** | カウントを一時停止します（開始中のみ有効） |
| **ラップ** | 現在の経過時間を記録します（開始中のみ有効） |
| **リセット** | すべてをリセットして00:00:00.00に戻します |

### 使用例

1. **基本的な計測**
   - 「開始」ボタンをクリック
   - 計測したい時間が経過したら「停止」ボタンをクリック
   - 「リセット」で初期状態に戻る

2. **ラップタイム計測**
   - 「開始」ボタンで計測開始
   - 記録したいタイミングで「ラップ」ボタンをクリック
   - 複数のラップタイムを記録可能
   - ラップは最新のものが上に表示されます

## 📋 技術仕様


- **言語**: HTML5, CSS3, JavaScript (Vanilla)
- **依存関係**: なし

## 🛠️ カスタマイズ

### 色の変更

`index.html` の `<style>` セクションで以下を変更できます：

```css
/* 背景グラデーション */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* ボタンカラー */
.start { background: #4CAF50; }  /* 開始ボタン */
.stop { background: #f44336; }   /* 停止ボタン */
.reset { background: #2196F3; }  /* リセットボタン */
.lap { background: #FF9800; }    /* ラップボタン */
```

### 計測精度の変更

JavaScriptの `setInterval` の値を変更：

```javascript
// 現在: 10ミリ秒（100分の1秒）
timerInterval = setInterval(updateDisplay, 10);

// 1ミリ秒精度にする場合
timerInterval = setInterval(updateDisplay, 1);
```
